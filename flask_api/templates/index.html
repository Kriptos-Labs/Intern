API rest to https://www.themoviedb.org/ API

For more information visit this link https://developers.themoviedb.org/3/getting-started/introduction


Requests:

/popular

List a popular movie

/search_by_id/'id-movie'

Search a movie by id

/create/'name_list'/'description'/'language'

Create a personal list in api

/view/'id_list'

View the list by id

/delete/'id_list'

Delete a list by id
